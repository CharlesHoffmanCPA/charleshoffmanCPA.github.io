Metapattern: RollForwardInfo (2017-05-07)

# Item Description
A. Metapattern name RollForwardInfo
B. Description A roll forward info is similar to a roll forward in that it has a beginning and ending balance; however, the numbers do not roll forward. The pattern is used to disclose the weighted average price rather than an amount. The data type is always num:perShareItemType.
